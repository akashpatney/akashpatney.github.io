2022 - Analytics Project:


Skills Used: C++, Visual Studio


Description: The purpose of this project was to construct algorithms that take input data to attempt to accurately predict the outcomes of basketball games. The project uses mathematical calculations to determine statistics strongly correlated with winning outcomes and to also compare respective teams in matchups. While I initally intended for this project to focus on the NBA, as I was developing the project, I realized that the program did not have to be limited to the NBA, but could really actually work for any basketball game. In the future, my hope is to be able to expand this project so that in addition to basketball, other sports are also added.
