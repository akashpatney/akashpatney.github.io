Express AR

Website

This project was the capstone project of my MDE class EECS 440 - Extended Reality for Social Impact. This app is aimed at promoting social emotional learning amongst children. The app includes two modes: story mode and express mode. In story mode, the user travels the room searching for characters, and unlocks them by guessing the emotion of the character correctly. In express mode, the user is presented with a prompt and tasked with making an expression that aligns with the scenario, which the facial recognition technology then assess. Correctly creating the expression awards the user points, which can be used for purchasing AR acessories.

Sports Analytics

GitHub

The purpose of this project was to construct algorithms that take input data to attempt to accurately predict the outcomes of basketball games. The project uses mathematical calculations to determine statistics strongly correlated with winning outcomes and to also compare respective teams in matchups. While I initally intended for this project to focus on the NBA, as I was developing the project, I realized that the program did not have to be limited to the NBA, but could really actually work for any basketball game. In the future, my hope is to be able to expand this project so that in addition to basketball, other sports are also added.
